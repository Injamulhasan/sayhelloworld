A modern agricultural ERP and farm intelligence platform designed to help farmers, agribusinesses, and exporters manage the complete crop lifecycle. 

Built with a Bengali-first user experience, it supports seamless language switching. The architecture features NextAuth.js for robust authentication, Prisma ORM with Supabase PostgreSQL for a cloud-ready database, and a highly interactive 11-step farm setup wizard with dynamic location mapping and state persistence.
