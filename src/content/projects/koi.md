A collaborative community coordination platform designed to manage group activities, location voting, event scheduling, and financial contributions.

The system utilizes a modern full-stack architecture leveraging React and Vite on the frontend, alongside an Express Node.js REST API. It integrates Supabase PostgreSQL for persistent storage and authentication, while utilizing WebSockets to provide real-time coordination and chat features.
